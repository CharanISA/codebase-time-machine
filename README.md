# Codebase Time Machine

Codebase Time Machine is a full-stack application that analyzes Git repositories and helps developers understand how a codebase evolved over time.

## Features

* Analyze public GitHub repositories
* View commit history
* Track top contributors
* Identify frequently changed files
* Visualize project evolution
* Generate project insights from repository activity

## Tech Stack

### Frontend

* React
* Vite
* CSS

### Backend

* FastAPI
* Python
* GitPython

## How It Works

1. Enter a public GitHub repository URL
2. The application clones the repository
3. Commit history is analyzed
4. Contributor statistics are generated
5. File change trends are calculated
6. Insights are displayed through an interactive dashboard
